# MedBook
Metodyka projektowania systemów informacyjnych (2026L)  

## Setup
1. Sklonuj repozytorium na swój komputer:
```bash
git clone https://github.com/RilleyPL/MedBook.git
```
2. Wejdź do projektu
```bash
cd MedBook
```


## Praca w zespole
1. Tworzenie brancha
```bash
git checkout -b z15_dodanie_wymagan_funkcjonalnych
```
2. Dodawanie zmian
```bash
git add .
git commit -m "Opis zmian, np. dodano moduł pacjenta"
```
3. Wysyłanie zmian na GitHub
```bash
git push origin z15_dodanie_wymagan_funkcjonalnych
```

## Aktualizacja głównej gałęzi
```bash
git checkout main
git pull origin main
```

sprawdzenie statusu
```bash
git status
```
